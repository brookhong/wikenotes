WikeNotes is a tool for someone to organize his notes in plain text or html. You can create a note, save it then find it by a keyword you typed.

  • to create a note
  1. just press ctrl+n or select File|New Note to jump into the edit form
  2. write your title/content/tag there or paste from clipboard
  3. press ctrl+s to save

  • to create a rich text(html) note
  1. just press ctrl+n or select File|New Note to jump into the edit form
  2. fill the title/tag, then write your html source code in the content edit
  3. then press alt+s to toggle view between html source and view
  4. switch back to source view, then press ctrl+s to save
